Projects

Airbnb Clone

This project is an Airbnb clone built with Express, Node.js, MongoDB, EJS, HTML, CSS, and JavaScript. It allows users to view, post, edit, and delete listings, add and delete reviews, and manage user authentication for sign-up and log-in. Users can search, filter, and sort listings by various categories and add new listings. The app includes a star-based rating system using Starability and integrates Cloudinary for image storage and MapTiler for displaying maps. Designed with Bootstrap, FontAwesome, and Google Icons, the website is fully responsive. The backend incorporates technologies like Mongoose, Passport, and Node Geocoder.

Simon Memory Game

The Simon Memory Game is a web-based game created with HTML, CSS, and JavaScript. Players start by clicking "START" and then watch a sequence of colors, which they must repeat in the same order. Each correct sequence advances the player to the next level, adding more colors. The game tracks the current and highest scores. A mistake ends the game, displaying a "Game Over" message and the final score. A reset option allows players to restart the game from the beginning.
